# Install-MY-SQL-Ubuntu

# how to install MYSQL on Ubuntu test on ubuntu 20.04 

1. sudo apt update

2. sudo apt upgrade

3. sudo apt install mysql-server

4. check mysql version
mysql --version

5. sudo mysql_secure_installation
